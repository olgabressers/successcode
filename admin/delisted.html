How to relist a delisted page

  1. Remove <meta name="robots" content="noindex, nofollow"> from the page <head>.
  2. Remove <meta name="googlebot" content="noindex, nofollow"> as well.
  3. Add a link to the page from a public-facing page (homepage nav, footer, or sitemap).
  4. Submit the URL to Google Search Console for re-indexing.
  5. Move the entry from this admin index to the appropriate public page reference.
  6. Verify with curl -I that the page returns 200 and the noindex header is gone.
